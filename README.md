# $MAGAHOOD — Digital Nation Economy

Futuristic Web3 landing site for **$MAGAHOOD**, built with React + Vite + Tailwind CSS.

## Primary color

**`#C6F700`** — neon green accent used across buttons, borders, glows, and highlights.

## Stack

- React 19
- Vite 8
- Tailwind CSS 4
- Framer Motion
- Lucide React

## Scripts

```bash
npm install
npm run dev      # local development
npm run build    # production build
npm run preview  # preview production build
```
